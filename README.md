# Sabor Local

Projeto acadêmico de delivery desenvolvido como parte do curso de Análise e Desenvolvimento de Sistemas.

## Objetivo

Criar uma solução digital para um restaurante local, permitindo que clientes façam pedidos, acompanhem entregas e utilizem recursos de fidelidade.

## Tecnologias e ferramentas

- FlutterFlow
- Xano
- APIs
- Banco de Dados
- Figma

## Funcionalidades

- Cadastro de usuário
- Cadastro de endereço
- Cardápio digital
- Pedido online
- Pagamento online
- Rastreamento de entrega
- Avaliação do pedido
- Programa de fidelidade
